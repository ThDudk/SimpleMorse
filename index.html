How to write morse code

Morse code is a telecommunications method which encodes text characters as standardized sequences of two different signal durations, called dots and dashes, or dits and dahs.

- wikipedia

Morse code is composed of two inputs. Dits and Dahs.

  • Dits last for one time unit.
  • Dahs last for three time units.

A letter is composed of dits and dahs, seperated by 1 time unit. For example, "a" in morse code is:

  • or ".-"

Words are composed of letters, seperated by three time units. For example the word "an" in morse code is:

  • or ".- -."

Finally sentances are composed of words, seperated by 7 time units. So the sentance "an ant" in morse code is:

  • or ".- -. / .- -. ..."
